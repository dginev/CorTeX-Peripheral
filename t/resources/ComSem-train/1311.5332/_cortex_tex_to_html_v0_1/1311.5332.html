Abstract.

1. Introduction

Given an nnn-vertex graph GGG, say that a set A⊆E⁢(G)AEGA\subseteq E(G) is triangle-independent if it contains at most one edge from each triangle of GGG, and say that X⊆E⁢(G)XEGX\subseteq E(G) is a triangle edge cover if G-XGXG-X is triangle-free. Throughout this paper, α1⁢(G)subscriptα1G\alpha_{1}(G) denotes the maximum size of a triangle-independent set of edges in GGG, while τ1⁢(G)subscriptτ1G\tau_{1}(G) denotes the minimum size of a triangle edge cover in GGG.

Erdős [] showed that every nnn-vertex graph GGG has a bipartite subgraph with at least |E⁢(G)|/2EG2\left\lvert{E(G)}\right\rvert/2 edges, which implies that τ1⁢(G)≤|E⁢(G)|/2≤n2/4subscriptτ1GEG2superscriptn24\tau_{1}(G)\leq\left\lvert{E(G)}\right\rvert/2\leq n^{2}/4. Similarly, if AAA is triangle-independent, then the subgraph of GGG with edge set AAA is clearly triangle-free; by Mantel’s Theorem, this implies that α1⁢(G)≤n2/4subscriptα1Gsuperscriptn24\alpha_{1}(G)\leq n^{2}/4.

Intuitively, α1⁢(G)subscriptα1G\alpha_{1}(G) and τ1⁢(G)subscriptτ1G\tau_{1}(G) cannot both be large: if τ1⁢(G)subscriptτ1G\tau_{1}(G) is close to n2/4superscriptn24n^{2}/4, then |E⁢(G)|EG\left\lvert{E(G)}\right\rvert is close to n2/2superscriptn22n^{2}/2, which makes it difficult to find a large triangle-independent set of edges. Erdős, Gallai, and Tuza formalized this intuition with the following conjecture.

Conjecture 1.1 (Erdős–Gallai–Tuza []).

For every nnn-vertex graph GGG, α1⁢(G)+τ1⁢(G)≤n2/4subscriptα1Gsubscriptτ1Gsuperscriptn24\alpha_{1}(G)+\tau_{1}(G)\leq n^{2}/4.

The conjecture is sharp, if true: consider the graphs KnsubscriptKnK_{n} and Kn/2,n/2subscriptKn2n2K_{n/2,n/2}, where nnn is even. We have

α1⁢(Kn)subscriptα1subscriptKn\displaystyle\alpha_{1}(K_{n}) =n/2,absentn2\displaystyle=n/2, τ1⁢(Kn)subscriptτ1subscriptKn\displaystyle\tau_{1}(K_{n}) =(n2)-n2/4;absentbinomialn2superscriptn24\displaystyle={n\choose 2}-n^{2}/4;

In both cases, α1⁢(G)+τ1⁢(G)=n2/4subscriptα1Gsubscriptτ1Gsuperscriptn24\alpha_{1}(G)+\tau_{1}(G)=n^{2}/4, but a different term dominates in each case. As observed by Erdős, Gallai, and Tuza, the difficulty of the conjecture lies in the variety of graphs for which the conjecture is sharp: any proof of the conjecture would need to account for both KnsubscriptKnK_{n} and Kn/2,n/2subscriptKn2n2K_{n/2,n/2} without any waste.

Erdős, Gallai, and Tuza [] considered the conjecture on graphs for which every edge lies in a triangle, and proved that there is a positive constant ccc such that α1⁢(G)+τ1⁢(G)≤|E⁢(G)|-c⁢|E⁢(G)|1/3subscriptα1Gsubscriptτ1GEGcsuperscriptEG13\alpha_{1}(G)+\tau_{1}(G)\leq\left\lvert{E(G)}\right\rvert-c\left\lvert{E(G)}% \right\rvert^{1/3} and α1⁢(G)+τ1⁢(G)≤|E⁢(G)|-c⁢|V⁢(G)|1/2subscriptα1Gsubscriptτ1GEGcsuperscriptVG12\alpha_{1}(G)+\tau_{1}(G)\leq\left\lvert{E(G)}\right\rvert-c\left\lvert{V(G)}% \right\rvert^{1/2} for such graphs. Aside from the original paper of Erdős, Gallai, and Tuza, no other work appears to have been done on the conjecture.

In this paper, we present two partial results towards Conjecture 1.1.

In Section 2, we extend some ideas of Erdős, Faudree, Pach and Spencer [] in order to obtain the bound α1⁢(G)+τ1⁢(G)≤5⁢n2/16subscriptα1Gsubscriptτ1G5superscriptn216\alpha_{1}(G)+\tau_{1}(G)\leq 5n^{2}/16. To our knowledge, this is the best general bound on α1⁢(G)+τ1⁢(G)subscriptα1Gsubscriptτ1G\alpha_{1}(G)+\tau_{1}(G).

In Section 3, we obtain the bound α1⁢(G)≤n2/2-msubscriptα1Gsuperscriptn22m\alpha_{1}(G)\leq n^{2}/2-m, where m=|E⁢(G)|mEGm=\left\lvert{E(G)}\right\rvert, and characterize the graphs for which equality holds. When nnn is even, this bound is sharp for both KnsubscriptKnK_{n} and Kn/2,n/2subscriptKn2n2K_{n/2,n/2}, which makes it an encouraging step towards the Erdős–Gallai–Tuza Conjecture.

2. Induced Bipartite Subgraphs

In this section, we will focus on the relationship between triangle-free subgraphs of GGG and bipartite subgraphs of GGG. The problem of finding a largest bipartite subgraph of a graph GGG is well-studied, and clearly any bipartite subgraph of GGG is triangle-free, so we can reasonably hope to apply some of the existing literature to our current problem.

Erdős, Faudree, Pach, and Spencer [] studied the problem of finding a largest bipartite subgraph of a triangle-free graph, using the observation that u⁢vuvuv is an edge of a triangle-free graph, then G⁢[N⁢(u)∪N⁢(v)]GNuNvG[N(u)\cup N(v)] is an induced bipartite subgraph of GGG.
